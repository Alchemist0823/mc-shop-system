Powermc
-------
Powermc is a responsive, HTML5 based, Zen 7.x-5.x sub-theme.


Installation
------------
To install Powermc:

- download both Powermc (latest version) and Zen 7.x-5.x
- unpack the theme folders ('powermc' and 'zen') and place them in your site's
  'sites/all/themes' directory (or other installation directory if required -
  see https://drupal.org/getting-started/install-contrib/themes)
- log into your site and go to the 'Appearance' page (/admin/appearance) - both
  Powermc and Zen should be listed
- click the 'Enable and set default' link for Powermc


Theme settings
--------------
Both Powermc and Zen include theme settings, allowing you to customise aspects
of your theme (e.g. layout).

You can find these settings by logging into your site and going to Powermc's
settings page (/admin/appearance/settings/powermc).


Documentation
-------------

- theme settings (including layout and max width)
- mobile friendly mini nav
- web fonts
- IE8 support
